# Scikit-learn Linear Regression Model for Ad Sales Prediction
📖 Project Overview

This project is a Machine Learning web application built using Scikit-learn and Streamlit.

It predicts Sales Revenue based on advertising budgets spent on:

📺 TV Advertising

📻 Radio Advertising

📰 Newspaper Advertising

The model is trained using Linear Regression from Scikit-learn.

🧠 Machine Learning Model

Algorithm: Linear Regression

Library: Scikit-learn

Model saved using: Pickle (.pkl)

The model learns the relationship between advertisement spending and sales performance.

💻 Web Application

The application is built using:

Streamlit

NumPy

Pandas

Scikit-learn

Users can input advertising budgets and instantly get predicted sales.
